# D&D AI 使用指南

这个仓库是一套给 AI 当 DM 用的 D&D 5e 风格工具集。  
核心 skill 是 `dnd-bg3-dm`，适合用来跑角色创建、战役开局、剧情推进、战斗结算、任务记录和长期存档。

## 怎么用

1. 打开这个仓库，把它作为 AI 的工作上下文。
2. 直接告诉 AI：

```text
请使用 dnd-bg3-dm 这个 skill，带我玩一局 D&D。
```

3. 继续补充你想玩的模式，例如：
   - 角色创建
   - 一次性短团
   - 持续战役
   - 纯规则问答
   - 战斗回合

## AI 会做什么

- 先问你想玩什么，再进入流程
- 按 D&D 5e 规则做裁定
- 记录角色、NPC、任务、地点、战利品和战役状态
- 在 `campaigns/` 里保存长期存档
- 需要时查询本地规则索引和设定资料

## 推荐玩法

### 角色创建

告诉 AI 你的角色概念，AI 会一步步带你完成：

```text
我想创建一个偏社交的半精灵游侠。
```

### 开始战役

```text
请新开一团原创奇幻战役，偏黑暗冒险风格，使用标准 5e 规则。
```

### 继续已有战役

```text
继续灰烬边境，帮我推进当前场景。
```

### 规则查询

```text
我想查一下专注和擒抱的规则。
```

## 这个仓库里有什么

- `SKILL.md`：AI 扮演 DM 的主说明
- `campaigns/`：战役存档
- `references/`：通用规则、设定和同步说明
- `rules/`：结构化规则与索引
- `control-panel/`：本地管理界面
- `scripts/`：索引构建和同步脚本

## 小提示

- 规则优先于即兴脑补
- AI 不会替你决定角色的重大行动
- 长团最好固定一个战役目录，方便后续续写
- 如果你启用 BG3 风格规则，要明确告诉 AI，不要默认混用

## 一句话指令

```text
使用 dnd-bg3-dm skill，按规则主持一场 D&D，我来当玩家。
```
