# 灰烬边境：战役剧本档案

> 这是本战役的追加式剧本记录。角色行动、DM回应、检定、规则裁定、经验和未解决选择都写在这里。

## 2026-07-16 - 车卡与开场

### 场景

凯德是一位旅行至灰烬边境镇的年轻人类吟游诗人，使用 PHB 2014 + SRD 5.1，原创奇幻世界，开放掷骰，紧张冒险并允许黑色幽默。

### 玩家

- 种族：人类
- 职业：吟游诗人 1 级；背景：骗子
- 属性掷骰最终结果：17、15、12、12、11、9
- 分配并应用人类加值后：力量 10、敏捷 16、体质 13、智力 12、感知 13、魅力 18
- 角色名：凯德
- 假身份：“伊尔明斯特，一位传奇大法师”
- 语言：通用语、精灵语
- 乐器：短笛、鲁特琴、号角
- 戏法：交友术、次级幻影
- 1 环法术：不谐低语、妖火、易容术、雷鸣波
- 装备：细剑、艺人背包、短笛
- 性格：永远诙谐幽默
- 理想：能够愚弄每一个人
- 羁绊：坚信自己是吟游诗人之神麾下最会讲笑话的人
- 缺陷：无论多么危急都不会抛弃幽默

### DM

凯德抵达灰烬边境镇南门。黄昏中，封锁多年的灰钟矿井冒出灰白烟雾。一名受伤矿工被抬到门口，守门人玛拉不愿让任何人靠近矿井。

玛拉： “如果你真想帮忙，就先让他把话说清楚。”

凯德向玛拉说明自己没有恶意，只是一名旅行至此、愿意排忧解难的吟游诗人。

凯德随后选择询问矿井历史。玛拉透露：矿井二十七年前发生东侧矿道坍塌，十七名矿工失踪；矿工们曾发现敲击会回应的黑色金属；封井后今晚第一次出现灰烟。

受伤矿工托宾说：矿井里有四个音，低、低、高、低；有一扇带无眼之脸的铜门；门后存在会模仿死者歌声的东西。

### 检定与规则

- 游说：`1d20+6 = 14+6 = 20`，成功。玛拉允许凯德调查，但要求谨慎。
- 奥秘：凯德检查托宾右耳的黑色裂纹，`1d20+1 = 16+1 = 17`，成功。判断为魔法共鸣线：它让托宾听见矿井，也让矿井听见托宾。
- 奥秘：凯德尝试切断共鸣线，`1d20+1 = 11+1 = 12`，失败。共鸣短暂锁定凯德，矿井听见并记住“伊尔明斯特”。

### 经验与奖励

- 暂不逐次结算；后续回溯结算时计入：与玛拉建立合作 `+25 XP`，查明共鸣机制 `+50 XP`，承受反噬并获得情报 `+25 XP`。

### 状态变化

- 玛拉：谨慎但合作。
- 托宾：仍与矿井保持共鸣，身体虚弱。
- 凯德：未受伤，但被矿井中的存在听见并标记。

### 当前状态

凯德、玛拉和托宾在灰钟矿井入口外。矿井已经用凯德的声音模仿“肺活量挺不错”。

### 下一选择

凯德可以用歌声或短笛回应四个音，继续询问托宾，检查矿井，或请求玛拉封锁现场。

## 2026-07-16 - 歌声开门

### 场景

凯德决定参加所谓的“音乐交流会”。玛拉愿意陪同，托宾表示自己可能必须同行，但最终留在入口附近。凯德判断自己身上的共鸣可能让自己成为开启铜门的钥匙。

### 玩家

凯德选择用歌声回应四个音：低、低、高、低。

### DM

矿井回应了同样的旋律。铁链震动，封门浮现无眼之脸。矿井用凯德的声音说：“请进，伊尔明斯特大法师。”

凯德与玛拉进入矿井。右侧矿道有新鲜脚印，只进不出；铜片上刻着“听见者，留下回声”。凯德询问矿井为什么重新开启，回声之主回答：矿井从未真正关闭，最后一块铜牌碎裂，凯德的歌让门重新记起开启方法。

### 检定与规则

- 表演：`1d20+6 = 18+6 = 24`，成功。凯德用自己的旋律变化让矿井困惑，获得进入权限。
- 察觉：`1d20+3 = 15+3 = 18`，成功。发现右侧脚印前重后轻、铜片和“听见者，留下回声”的提示；右侧无面之物开始接近。
- 欺瞒：凯德宣称自己是传奇大法师，`1d20+6 = 7+6 = 13`，失败。回声之主看出他在撒谎，但对他感兴趣。

### 经验与奖励

- 歌声开启矿井：`+50 XP`。

### 状态变化

- 回声之主自称“回声之主”，但承认这只是称号而非真正名字。
- 凯德被称为“借名者”；矿井对他的声音和假身份高度敏感。

### 当前状态

凯德与玛拉进入矿井左侧坑道。老霍尔在尽头的圆形矿室等待，重复前三个音并寻找最后一个音。

### 下一选择

询问老霍尔、追问铜门后的东西、唱出最后一个音，或调查坑道和矿室。

## 2026-07-16 - 老霍尔与最后一个音

### 场景

老霍尔被发现坐在矿室中央。他自称是二十七年前的矿工，但身体已经由回声拼成。老霍尔解释：门后不是房间，而是“回声室”；被埋在矿脉下的存在只有耳朵、喉咙和部分记忆，会通过听见别人来拼出自己。

### 玩家

凯德先用贵族礼仪质问回声之主的名字，后询问矿井重新开启的原因。面对老霍尔时，他说自己只是“一无所有、穷困潦倒、来混口饭吃的吟游诗人”。

凯德随后表示要展示看家本领，把四个音扩展成歌谣，加入自己来到镇上的所见所闻。

### DM

凯德的歌谣唱了灰烬边境镇南门、玛拉、托宾、灰烟和一个假装传奇大法师的穷困吟游诗人。凯德没有补上回声期待的第四个音，而是创造了第五个、无法预测的结尾。

### 检定与规则

- 欺瞒：`1d20+6 = 17+6 = 23`，成功。老霍尔相信凯德不是为了力量而来，但仍认为他是矿井的钥匙。
- 表演：`1d20+6 = 19+6 = 25`，大成功。凯德改写最后一个音，切断自身共鸣，令回声之主无法直接从门后走出。

### 经验与奖励

- 与回声之主、老霍尔交涉并查明回声室性质：`+50 XP`。
- 改写歌谣结尾并切断自身共鸣：`+100 XP`。
- 本段回溯结算合计：`+300 XP`。
- 经验条：升级前 `0 / 300 XP`；获得后 `300 / 300 XP`，凯德升至 2 级。
- 升级后：HP `15/15`，生命骰 `2d8`，1 环法术位 `3`，获得万事通和休憩曲 d6。
- 2 级新增法术：玩家选择**治愈真言**。
- 当前经验：`300 / 900 XP`；距离 3 级还需 `600 XP`。

### 状态变化

- 凯德身上的共鸣线消失。
- 老霍尔恢复部分自我，留在上层矿室。
- 回声之主仍存在，但被凯德的新结尾暂时压回黑暗。
- 凯德随口声称自己是矿井掌控者、回声之主只是仆人。矿井将这句话记录为临时规则。

### 当前状态

凯德和玛拉位于地下大厅。大厅中央有无舌铜钟、黑色镜子和三扇门：

1. 左门：十七名矿工的残留回声。
2. 中门：正在模仿凯德旋律的鲁特琴。
3. 右门：回声之主的黑暗阶梯，已称凯德为“主人”并等待第一条命令。

凯德刚刚对三扇门后的存在说：“请告诉我你们的来历和目的。”

### DM

三扇门共同回答：十七名矿工死于坍塌或被留下；它们是被记住的声音；回声之主是第一个完整的回声。它们想离开矿井，获得身体和名字。回声之主说：“凯德，你已经给了我们一个结尾。现在，请给我们一个开始。”

玛拉问：“掌控者大人，你准备怎么处理你的‘仆人’？”

### 当前状态

- 地点：灰钟矿井地下大厅。
- 时间：2026-07-17，黄昏后。
- 凯德：2 级吟游诗人，HP `15/15`，AC `14`，经验 `300/900 XP`，1 环法术位 `3/3`。
- 玛拉：同行，警戒回声实体。
- 老霍尔：留在上层矿室，暂时恢复自我。
- 托宾：留在矿井入口附近，身体虚弱。
- 经验结算规则：每次获得经验必须显示本次 XP、当前经验/下一级门槛、距离升级。

### 下一选择

凯德需要决定第一条命令：要求回声说明更多规则、命令它们不得夺取活人身体、探索左门的十七个名字、调查中门的鲁特琴、进入右门的黑暗阶梯，或继续用玩笑与谈判试探矿井。
