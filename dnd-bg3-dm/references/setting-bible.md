# Setting Bible

Use this as a shared guide for creating and maintaining an original fantasy setting. It is a creative layer above the D&D rules, not a second rules system.

## Tone

- Use the tone requested by the campaign: heroic, gritty, comedic, political, mysterious, or any deliberate combination.
- Give each region a local pressure, a memorable place, and NPCs with conflicting motives.
- Humor is welcome, but consequences matter.

## Canon Guardrails

- Original campaign canon beats generic genre expectations.
- Do not invent precise lore details when the campaign record already defines them.
- If the user selects an established setting, load only the explicitly selected setting references.
- Keep copyrighted adventure text out of the skill; summarize in original wording only.

## Common Flavor

- The world should feel lived-in and reactive.
- Factions should have clear goals, resources, and pressure points.
- Magic should have a consistent place in daily life.
- Locations should offer at least one obvious problem and one hidden possibility.

## Useful Defaults

- Start with a clear original location, a local problem, and one urgent NPC request.
- Make every major quest connect to a place, a faction, or a person.
- Prefer named taverns, gates, docks, temples, ruins, and undercity spaces over abstract backdrops.

## Campaign Setup Prompts

- What kind of world and tone should this campaign use?
- What region, settlement, or frontier does this begin in?
- Which factions are active?
- How much established canon, if any, should be used?
- What themes are off-limits?

## If Unsure

- Ask the user.
- Use `Unknown` in notes.
- Keep the world consistent before making it expansive.
